# street-fighter-pygame

# CONTROLS SO FAR

## PLAYER 1

WASD: movement

R: light attack (punch)

T: heavy attack (kick)

## PLAYER 2

ARROWS: movement

N: light attack (punch)

M: heavy attack (kick)

punch does 10dmg, 30 frame cooldown
kick does 20dmg, 100 frame cooldown

punch punch kick seems to be the highest DPS combo;)
