Welcome to Martial Mayhem, the online multiplayer fighting game made with pygame

Getting Started.

 

There are two ways to play!!

  • 1. Single player and local player are available from this website in the browser
  • 2. All game modes including multiplayer are available through the download link on this website or the project github

 

Option 1: Playing on this website.

All you have to do to begin playing the game in single player is click on the game link at the top of this page. you will have two options available to you single player and local player where you and your friends can play on the same keyboard!! But to play in multi-player you will have to download the game to do this follow the steps for option 2.

 

Option 2: Download from this website or from our GitHub.

To download from this website simply click the download link at the top of the page or click here Download If you would prefer to download from our GitHub please follow the following steps.

  • 1. Go to your favourite terminal, clone the repository at this link github.com/michaelc445/street-fighter-pygame
  • 2. Enter the following commands:
    • pip3 -m install requirements.txt
    • python3 main.py
  • 3. in main.spec, change the path in line 9 to the path where your python packages are located. and be sure to use double backslashes
  • 4. in the terminal where main.spec is located type: pyinstaller main.spec

 

How to play

  • - Select your Fighter and the map
  • - Win rounds by attacking the opponent until their health bar is depleted
  • - First fighter to win 3 rounds wins the game!

 

Game Modes

  • Battle against the computer in an intense one on one match!
  • local Play your friend locally using the same keyboard!
  • multiplayer Queue up for an intense match online!
  • We use a server provided by NetSoc, which can host up to 10 simultaneous games.
